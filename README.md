# Solar Panel Defect & Cleanliness Detection

This project detects surface irregularities and cleanliness defects on solar panels using OpenCV image processing techniques such as edge detection and blob (spot) detection.

It helps renewable energy firms automatically identify defective or dirty regions on solar panels and quantify defect severity.

## 🎯 Objective

- Detect surface defects and dirt spots on solar panels.
- Calculate defect area percentage.
- Classify severity based on detected regions.
